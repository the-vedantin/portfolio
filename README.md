# Aayu Yadav Portfolio Migration

This is a static migration of the Google Sites portfolio at `https://www.aayuyadav.info/home`.

## Files

- `index.html` is the page shell.
- `content.js` stores migrated content, links, image URLs, CAD links, and embeds.
- `app.js` renders the current page from the URL hash.
- `styles.css` provides a clean baseline UI that can be redesigned without touching content.

Open `index.html` directly in a browser, or serve the folder with any static server.
